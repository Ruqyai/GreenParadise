# GreenParadise

A smart waste container connected to an application on the user phone and with the waste management website. This container can distinguish the type of waste and give the user a bonus if the waste is thrown in the designated area. 

### It is a part of Green Paradise project which is a participant in Hajj Hackathon.  
### For more information please click  [here](https://devpost.com/software/n-030-green-paradise).
